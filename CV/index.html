A credits management system devoloped as a group 2. Semester Project
February, 2021 - May, 2021
The software solution has been developed using Java, SQL and JavaFX for the User Interface. The project consisted of developing a credits management system for the danish tv station TV. The software had the following requirements: Create, edit, delete and save new credits for movies, tv series etc. Search trough a database of these credits and display them through a simple UI to the user. A key part of the project was learning different software development strategies and how to work as engineers in general. This project was developed using a mix of Unified process and SCRUM. Developing the code was actually a minor part of the project as we focus on getting requirements, diagrams and analysis finished before any actual coding was done.
